# discover-skills

> Describe what you need in plain language, find the right skill instantly.

As AI agent skills grow rapidly, finding the right one isn't always easy.

**discover-skills** deeply vectorizes every skill, so you can describe your task in natural language and get the most relevant results.

## Why discover-skills?

- **Semantic Search**: Not just keyword matching. We understand your intent.
- **Fine-grained Indexing**: Every skill is analyzed and vectorized across multiple dimensions for better semantic understanding.
- **Multi-tool Support**: Works with Claude Code, OpenCode, Codex CLI, Gemini CLI, and more.

## How to Use

### Option 1: Search on Website

Visit [skillradar.quest](https://skillradar.quest) and describe your task. Get instant results with rankings and one-click install commands.

### Option 2: Install This Skill

Install `discover-skills` directly into your AI agent, and let it find skills for you automatically.

**Quick Install:**

Copy the install command from [skillradar.quest](https://skillradar.quest) homepage and paste it into your AI agent.

Once installed, your AI agent will automatically discover and suggest relevant skills when needed.

## Advanced Options

When searching, you can specify:

- **Task Constraints**: e.g., "must run locally", "output as JSON"
- **Keywords**: domain-specific terms to narrow results
- **Max Results**: control how many candidates to return
