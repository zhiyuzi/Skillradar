# discover-skills

> 用自然语言描述需求，精准找到你要的技能。

AI Agent 技能越来越多，找到合适的那个并不容易。

**discover-skills** 对每个技能做了深度向量化索引，你可以用正常的语言描述任务，我们会理解你的意图并返回最匹配的结果。

## 为什么使用 discover-skills？

- **语义搜索**：不只是关键词匹配，我们理解你的意图。
- **精细索引**：每个技能都经过多维度分析和向量化，更好地理解语义。
- **多工具支持**：支持 Claude Code、OpenCode、Codex CLI、Gemini CLI 等主流工具。

## 如何使用

### 方式一：官网搜索

访问 [skillradar.quest](https://skillradar.quest)，描述你的任务，即刻获得带排名的结果和一键安装命令。

### 方式二：安装本技能

将 `discover-skills` 直接安装到你的 AI Agent 中，让它自动为你发现技能。

**快速安装：**

从 [skillradar.quest](https://skillradar.quest) 首页复制安装命令，粘贴到你的 AI Agent 中执行即可。

安装后，你的 AI Agent 会在需要时自动发现并推荐相关技能。

## 高级选项

搜索时可以指定：

- **任务约束**：如"必须本地运行"、"输出为 JSON 格式"
- **关键词**：领域相关的术语，缩小搜索范围
- **结果数量**：控制返回多少个候选技能
